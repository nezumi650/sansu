# sansu
https://nezumi650.github.io/sansu/

算数の練習用サイトです。

パソコン、スマートフォン、タブレットから操作できます。

⚠️スマートフォンからの操作時は画面が小さいです。

⚠️操作中、効果音が鳴ります。
